# Snake Game

Project Description
This is a browser-based Snake game where the player controls a snake to collect food. Each time the snake eats food, it grows longer and the score increases. The game ends if the snake collides with the wall or itself.

 Technologies Used (Visual Studio Code)
- HTML
- CSS
- JavaScript (Canvas API)

 Setup Instructions
1. Clone or download the repository
2. Open index.html in a browser

How to use 
- Use arrow keys to control the snake
- Eat food to grow and increase score
- Avoid hitting walls or yourself
